URL Shortener API

A FreeCodeCamp exercise

Route HTTP Verb Description
/ GET Shows this page
/:url GET Many things can happen. Check below for details.

Here is how :url is processed

  • Check if it is already saved in database as original url:

    In that case returns a JSON containing both the original and shortened url

  • Is it saved as shortened url?

    You've just been redirected!

  • Is it a valid url?

    The shortened url is generated and saved in database. A JSON is returned.

  • It isn't a valid url.

    Nothing to be done really, a JSON with an error is returned.

Returning JSON

  • Everything was smooth: { "originalUrl": [String], "shortenedUrl": [String] }
  • An error occurred: { "error": [String] }

Example usage

Input

https://fcc-shortenerurl.herokuapp.com/https://www.freecodecamp.com/lbpinheiro

Result:

{"originalUrl":"https://www.freecodecamp.com/lbpinheiro","shortenedUrl":"http://fcc-shortenerurl.herokuapp.com/j9vwpy"}

Input

http://fcc-shortenerurl.herokuapp.com/j9vwpy

Result:

Redirect to https://www.freecodecamp.com/lbpinheiro

Input

http://fcc-shortenerurl.herokuapp.com/fu7733___2

Result:

The :url parameter is not a valid url neither a shortened url

{"error":"not a valid url"}
